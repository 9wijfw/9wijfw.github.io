Python3教程(二）---变量与常见数据类型

2025-01-10 python

欢迎阅读收看我的博客
变量:
您可以把变量立即为一个用于存储数据的”盒子“，这个盒子可以存下任何东西且可以被改变。
例如下行中的x就是一个变量。

- x = 514
- x = 114(x的值被改变)

如果我们想改变数据中的内容我们可以另起一行在x变量等号右侧输入新的内容那么它就会被改变
变量命名规则：在python中变量不可以用用数字或特殊符号开头(如!<@#$%等)，不可以使用python的关键字(例如自带的函数，布尔值等),变量最好命名清晰以便维护（当然这一点取决于你的个人习惯）

- #正确例子
- a =114514
- a1A = 114515
- _aaa_b = "113"
- 定 = 112(中文仅限python3.x)
- #错误例子
- 5A = "Q13"
- qq+11  = 143
- q a = 114*

常量：通常以大写字母表示且内容不会改变（当然这是人为规定的，如果想上文的方式去改变也不会报错)

- HEIGHT = 1080

值得注意的是，当一个变量里什么都没有时，我们用None表示
数据类型(括号内为这个数据类型的英文名):
#注意: python中数据的排布是从0开始的(即：一个字符串'rgfsd'的第0项是r,第一项是g)
1.整数(int)
这类数据顾名思义是一个整数 (听君一席话如听君一席话)
整数是由正整数，0，负整数的整数构成。在变量中，可直接与其他整数或浮点数进行加减乘除运算(ps:在相除的情况下最后算出来是浮点数)，这些数通常没有""存在，如果你在程序中看见如"1","123","14"等类似于这样形式存在的"整数"时，其实他们并不是整数，而是字符串。

- a = 1
- b = a+a --->b = 2
- c = 2*a ---> c = 2
- d = a/1 --->d = 1.0<class 'float'>

2.字符串(str)
字符串可以理解为存储文字的一种，通常以"xxxx"的方式存在，在变量中，把一个字符串与另一个字符串相加可把两段文字拼在一起(如果你您使用其他运算符会报错)。

- a = '1'
- c = ''#这是一个空字符串
- b=  '2'
- print(a+b) ---> '12'

3.浮点数(float)
浮点数就是我们俗称的小数(例如:0.114),这个数据类型的规则和整数类似。一般情况下，一个浮点数与一个整数相加，最后所得的结果永远是一个浮点数。
4.列表(list)
列表与前面所介绍的字符串和整数不同，一个列表可以同时存储多个整数，浮点数，字符串，列表等，外面用中括号括起来

- list_1 = [0,'1',334.'非结构',2.1]
- list_kong = []#这是一个空列表

5.字典(dict)
字典的特点是每一项都有一个键值对存在，即xxx:yyy为一项(xxx为键，yyy为值,且键可以填入不可被改变的数据类型、值可填入所有数据类型),外面由花括号包裹

- dict_1 = {'a':12,'b':333}
- dict_2 = {}#这是一个空字典

6.元组(tup)
虽然元组的形态与列表有几分相似，但是它的外部是由小括号括起来的。元组内部的元素不可被修改，但是它可以填入任何类型的数据（你甚至可以套一个列表，集合，字典和元组)，且两个元组相加可得到一个新元组。
#ps :元组内不可只存在一个整数

* tup1 = ([1,2],1)
* t = (12,1)
* print(tup1+t) --->([1, 2], 1, 12, '1')

7.集合（set)
它是由大括号进行包裹，内部和列表和元组类似，运算规则也与高中数学的集合相似

- set_1 = {1,4,5,6,7}

8.布尔值
这个数据是用来判断真(True)或假(False)时使用,它只有两个值True和False

 - a = True
