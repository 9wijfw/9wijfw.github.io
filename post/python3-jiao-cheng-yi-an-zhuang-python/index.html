Python3教程(一）---安装Python

2024-12-01 python

欢迎大家访问我的博客，从今天起我将发布有关python教学的内容
#注意：本教程只适用于Windows系统
python安装第一步需要访问python的官网：https://www.python.org/（见下图）

然后把光标移向"Downloads"位置（见下图标红的位置）

您可以点击在右侧的灰色按钮立即下载最新版本的python，也可以点击标黑箭头的位置寻找其他版本下载
(下图展示的是其他版本的下载界面）

我们就以python3.8.1版本为例：
在我们点击下载后会把我们带到另一个页面，在这个页面往下翻会找到一个以"Files"为标题的模块，下面就是下载链接（根据电脑的不同请自行选择下载适合的安装包）(https://9wijfw.github.io/post-images/1734188399624.png)
在点击"Windows x86-64 可执行安装程序"将会下载安装包(通常下载时间较长，请耐心等待)
下载完成后打开文件
1代表默认安装
2代表自定义安装(会让你选择安装路径<这个路径里最好不要有中文>，选装pip等)
图片下面画圈的最好都要勾上（尤其是第二个，不需要你在配置环境变量）

到了这一步算是安装完成了（Diable path length limit 是取消路径长度限制的可以不用管）,点击close即可
